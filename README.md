# Hospitality-Revenue-Insights-Dashboard

## Project Overview
This project is a Power BI dashboard built to analyze revenue performance in the hospitality domain.  
It provides insights into key business metrics like occupancy, revenue, booking trends, and customer behavior.

The dashboard helps stakeholders make data-driven decisions to improve profitability and operational efficiency.

---

## 🎯 Business Objectives
- Analyze hotel performance across cities and properties
- Track revenue and occupancy trends
- Identify booking patterns and customer behavior
- Monitor cancellations and no-show rates
- Evaluate platform-wise and room class performance

---

## 📈 Key Metrics (KPIs)
- Revenue
- RevPAR (Revenue per Available Room)
- ADR (Average Daily Rate)
- Occupancy %
- Realisation %
- Cancellation %
- No Show Rate %
- Total Bookings
- Average Rating

---

## Dashboard Features
-  City & Room Class Filters
-  Weekly Trend Analysis (WoW Change %)
-  Property-level performance comparison
-  Revenue & Occupancy trend visualization
-  Booking distribution by platform & category
-  Customer rating insights

---

## Key Insights
- Luxury category contributes higher revenue (~61%)
- Weekends show higher occupancy compared to weekdays
- Some properties have high revenue but low occupancy → pricing strategy issue
- Cancellation rate is significant (~24%) affecting overall revenue
- Direct online bookings show better realization %
